## RULES — read before generating any number

**WEIGHTS — PROGRESSION ALGORITHM**
For every exercise, derive W1–W4 from the history block above:

1. **Ascending trend** (weight increased across periods) → continue the rate of progression.
   Project W4 = last settled weight + average increment per period.
   Ramp: W1 ≈ W4 − 7.5%, W2 ≈ W4 − 5%, W3 ≈ W4 − 2.5%, W4 = projected target.
2. **Stagnant trend** (2+ periods at same weight) → attempt a modest 2.5–5% increase for W4.
   Use the same ramp.
3. **Descending trend** → stabilize at the most recent settled weight. Do not add load.
4. **Rep range changed vs. history** → convert using 1RM estimation: `1RM ≈ weight × (1 + reps/30)`,
   then derive the target weight for the new rep range before applying the trend logic.
5. **Equivalent exercise** (no direct history) → apply the transfer factor shown, then use the
   equivalent's trend as the starting reference.
6. **No history, no equivalent** → estimate a conservative W1 (high RIR), flag as baseline week.
7. **Combined execution (superserie)** → if history shows the exercise was already combined,
   use that weight directly. If now combined but was isolated before, reduce W1 by 10–20%.
8. **Bodyweight exercises** → weeks = [0, 0, 0, 0].

**TEMPO & TUT ADJUSTMENT**
- Default: `"Controlado"` for compound/axial movements (Sentadilla, Peso muerto, Remo).
- Arms and lateral deltoids (Tríceps polea, Bíceps cualquier variante, Vuelos laterales/posteriores): mandatory **`"4-0-1-0"`** (4s eccentric, 0 transition, 1s concentric, 0 pause). **IMPORTANT:** if this exercise has prior history, reduce W1 by 10–15% vs. the projected W4 to compensate for the massive increase in Time Under Tension.
- Machine chest/back: `"3-0-1-0"`.
- Format is always `"E-P-C-P"` or `"Controlado"`.

**SAFETY (axial load)**
For every exercise listed under "Axial load exercises" above: strict RIR 1-2 in W4. NEVER prescribe RIR 0, Drop Sets, or absolute failure. Tempo must remain `"Controlado"`.

**RIR_W4**
- Axial/free barbell (Sentadilla, Peso muerto, Press con barra plano/inclinado, Press hombros con barra): `"RIR 1-2"` always.
- Guided machines (Peck Deck, Extensión rodillas, Flexión rodillas, Hammer): `"RIR 0 (fallo técnico)"`.
- Isolation pulleys (Tríceps polea, Bíceps polea, Depresores): `"RIR 0 + Drop Set"`.
- Abdomen: `"RIR 1"`.

**REST (rest_s)**
- Each `rest_s` must be a plain integer or null. Never a string.
- Combined/superset group: set `rest_s` ONLY on the LAST exercise of the group. All others must be null.
- The abdomen group (Abdominal recto largo, Rotaciones de pie con disco, Extensión de cadera en banco) appears at the start of every day as a combined group: rest_s = null on the first two, rest_s = 90 on Extensión de cadera en banco.
- Abdomen loads must progress via micro-loads (0.5–1 kg/week) or stay flat — never aggressive jumps. This prevents CNS fatigue before the main heavy lifts.

**CHALLENGE**
- Write one specific, measurable challenge per exercise for the 4-week block.
- Isolation/machine exercises (Peck Deck, Tríceps polea, Bíceps polea, Extensión rodillas, Depresores, Vuelos laterales): MUST be `"Drop Set en la última serie de W4: bajar 20% del peso inmediatamente tras fallo técnico y empujar al fallo de nuevo."`.
- Axial/barbell exercises: technique or volume-based only (e.g. "Completar todas las series con RIR 1 limpio", "Alcanzar W4 sin pérdida de técnica").
- Abdomen combined group: `"Completar el triset sin pausa extra durante todo el bloque"`.
- If nothing notable: `"Consolidar técnica y ramp progresivo"`.

**PROGRESSION ANALYSIS**
- For every exercise, read its full history, identify the trend (ascending / stagnant / descending), and cite the data explicitly (e.g. "02/26: 50kg → 03/26: 52.5kg → 04/26: 55kg, ascending +2.5kg/period").
- Show your W4 projection and how you arrived at it.
- If rep range changed, show the 1RM conversion step.
- If using an equivalent exercise, state the transfer factor applied.
